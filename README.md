## 🔎 Overview
A basic END-TO-End Analyze a retail store 2023 data sales analysis in SQL
### 📊 Dataset
##### Table :- 
- Retail_Store 
##### Columns :-
- Transaction Id
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price Per Unit
- Total Amount 
### 🛠️ Tools Used
- Postgre SQL
- PG Admin4

- Customer Id rank by the Amount
